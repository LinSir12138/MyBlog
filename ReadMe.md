# 博客简介

基于SpringBoot+Thymeleaf的个人博客，使用MyBatis整合数据库，使用Redis作为缓存；用ElasticSearch实现全局搜索并高亮，使用使用七牛云的OSS对象对象存储服务；使用Edit.md做Markdown编辑器；发送邮件时使用RabbitMQ作为消息队列。
